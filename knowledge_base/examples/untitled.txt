Usage Examples:
1. Index all documents:
bashcd workspace/Google-STT-Google-TTS
python knowledge_base/examples/index_documents.py
2. Reset index and re-index:
bashpython knowledge_base/examples/index_documents.py --reset
3. Index documents and run tests:
bashpython knowledge_base/examples/index_documents.py --test
4. Just run tests (without indexing):
bashpython knowledge_base/examples/index_documents.py --test-only
5. Interactive query mode:
bashpython knowledge_base/examples/index_documents.py --interactive
6. Show current index stats:
bashpython knowledge_base/examples/index_documents.py --stats
